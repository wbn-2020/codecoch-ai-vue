3. 面试创建 / 行业场景 / 面试房间 / 报告。

## 当前风险

- `/admin/files` 后端仅提供分页和详情；没有 admin 删除、下载、重试解析接口。
- 文件列表后端不支持文件名关键词查询；当前仅支持 userId、bizType、status。
- Token 失效当前仍走清理本地登录态并跳转登录页；产品闭环依赖重新登录。
- 大 chunk 和 `@vueuse/core` PURE 注释仍为构建 warning，不影响 build。
